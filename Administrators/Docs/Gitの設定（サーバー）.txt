基本的に
http://mattintosh.blog.so-net.ne.jp/2012-09-14
の手順の通り実行

※ Git は
https://code.google.com/p/git-core/downloads/list
から原則最新のものをダウンロード…なのだが、
1.8 は gmake でエラーが出てしまうので、1.7 の最新 1.7.12.3 をダウンロードした。

抜粋：
▽▽▽
bash
mkdir -p $HOME/local/src
cd $HOME/local/src
wget http://git-core.googlecode.com/files/git-1.8.5.3.tar.gz
tar zxvf git-1.7.12.3.tar.gz
cd git-1.7.12.3
./configure --prefix=$HOME/local
gmake
gmake install
export PATH=$HOME/local/bin:$PATH
△△△

その後コマンドパスを通す
%vi .cshrc
set path = (〜) の () 内に
 $HOME/local/bin
を追加

puttygen.exe で秘密鍵を生成 ---
OpenSSL の公開鍵
openssh_pub_key.txt
を FTP などで自分のディレクトリ（/home/kishinet/）の .ssh ディレクトリにコピーする。
putty で kishinet.sakura.ne.jp にログインする。
% cd .ssh
% chmod 700 ./
% cat openssh_pub_key.txt >> authorized_keys
% chmod 600 authorized_keys
% rm openssh_pub_key.txt
