◆文と行

Java には行という概念は（行コメント以外に）基本的にありません。
プログラムは（コメントを除いて）文で構成されています。
改行は単語やリテラル、演算子の間に自由に入れることができます。
そのため、全てを１行に書くことも可能です。
（ただし、行コメントがないこと。）

この性質を利用して、プログラムをわざと簡単に読めないようにすることもあります（難読化）。


◆コメント

ブロックコメント
/* ・・・ */
/* から */ までがコメントアウトされる

行コメント
// ・・・
// から改行までがコメントアウトされる
