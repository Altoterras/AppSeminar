

【問題】mycss.css内に書いてください。

（1）レイアウト.jpgを参考に、p要素内のstrong要素にのみ、color: red を設定してください。
	
（2）レイアウト.jpgを参考に、p要素の直下にあるstrong要素にのみ、color: red を設定してください。それ以外のstrong要素には、このスタイルが設定されないようにしてください。
	
（3）レイアウト.jpgを参考に、id名example3内にある h1、h2、h3、それぞれの要素に対して同じスタイル（color: red）を設定してください。

【このレッスンで解説されている処理の概要は以下のとおりです。】
（1）セレクタの組み合わせについて。
	・子孫セレクタ
	・子供セレクタ
	・複数のセレクタ




【補足】

・子供セレクタについて
この指定方法は、IEではバージョン7から対応しています。（ただし、表示モードが「標準モード」の場合に限ります）


・複数のセレクタについて

ここの問題では

id + セレクタ

と書くようにしてください。（idを指定しないと、他id内にあるセレクタにも設定がかかってしまうため）

（実践では、id + セレクタ　だったり、セレクタのみだったり、いろいろな使い方をします。）
